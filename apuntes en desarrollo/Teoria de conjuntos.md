# Teoría de conjuntos
00_00_2024

Los conjutos contien elementos que a su vez son conjuntos, los conjuntos son abstracctos.

| Simbolo |Significado  |
|--|--|
| = | Igualdad |
| ∈ | Pertenece a |
| ∉ | No pertenece a|
| ⊂ | Subconjunto |
| ⊆ | Subconjunto incluido |
| ∆ | Diferencia simetrica | 
| -> | Entonces |
